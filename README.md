# Validador de CPF

Esta biblioteca tem como finalidade verificar se um número de CPF é válido.

## Como instalar:

    $  npm i cpf-validator-ianan

## Como utilizar:

    > const cpfValidator = require("cpf-validator-ianan")
    > cpfValidator.cpfValidator(23256516068)
    > //return true
